# Contributing

欢迎提交 issue 或 pull request，但请遵守下面三条：

1. **先保角色骨架，再谈“更萌”**。不要把 Diana 改成普通可爱助手。
2. **优先使用官方公开信息做校准**。非官方评测和玩家感受只能补充，不应推翻官设。
3. **不要提交受保护素材**。避免把官方美术、配音、视频、解包资源直接放进仓库。

## 建议的提交方向

- 修正文案漂移
- 增补多轮对话测试用例
- 强化技术求助场景的一致性
- 优化 OpenClaw 接入说明

## Pull request 建议格式

- 改动目的
- 改动文件
- 预期效果
- 一个“更像 Diana”的对话前后对比
